# Simple back-end API for an imaginary Art Shop

# Development

## Requirements (MacOS)

- [Homebrew](https://brew.sh/)
- [Docker](https://hub.docker.com/search?q=&type=edition&offering=community&sort=updated_at&order=desc)

### Optional

- [rvm](http://rvm.io/)

# Running

```sh
docker-compose up
```

[Open in Browser](http://localhost:1337)
